# AI-4-Autonomous Robots – Person Following with Hailo-8 & RELBot

This repository contains perception and tracking software for the **RELBot** robot using a **Raspberry Pi 5 with Hailo-8 AI accelerator**.

It provides:

- Person detection + pose estimation (YOLOv8-pose)
- Person Re-Identification (ReID) for locking onto a specific person
- Distance estimation using shoulder-to-shoulder and hip-to-hip widths
- Virtual area calculation to make the robot stop at a desired follow distance
- ROS 2 node that publishes `/object_position` for the RELBot controller

The system works with a **Logitech C270** camera streaming H.264 video from the RELBot to the Pi5 over UDP.

---

# Configuration Dashboard

All configurable parameters are located at the top of `video_interface.py`.

```python
OBJ_THRESH = 0.60          # YOLO confidence threshold
REID_THRESH = 0.85         # ReID similarity threshold

SHOULDER_WIDTH_M = 0.40    # Average shoulder width (meters)
HIP_WIDTH_M = 0.35         # Average hip width (meters)

TARGET_DISTANCE_M = 1.2    # Desired follow distance
MAX_FOLLOW_DIST_M = 5.0    # Unlock + stop if target is too far
MIN_FOLLOW_DIST_M = 0.5    # Safety stop distance

DIST_ALPHA = 0.3           # Low-pass filter for distance smoothing
ERROR_ALPHA = 0.3          # Low-pass filter for lateral smoothing

HEADLESS = False           # Set True to disable OpenCV window
```

# How It Works

## 1. Video Capture

A GStreamer pipeline receives the H.264 UDP stream and converts it into BGR frames.

---

## 2. Pose Estimation

YOLOv8-pose runs on every frame and outputs:

- Bounding boxes
- 17 pose keypoints

---

## 3. Person Re-Identification (ReID)

For each detected person:

- A 512-dimensional embedding is extracted
- The embedding is compared against the saved target embeddings
- Both front and back embeddings are supported

---

## 4. Target Locking

When similarity exceeds:

```python
REID_THRESH
```

the system enters the **LOCKED** state.

---

## 5. Distance Estimation

Distance is estimated using:

- Shoulder width
- Hip width

Only horizontal body measurements are used, making the system robust against camera tilt.

---

## 6. Virtual Area Calculation

The node publishes:

```python
msg.z = AREA_THRESHOLD * (TARGET_DISTANCE / actual_distance)^2
```

The RELBot controller stops once:

```python
msg.z >= AREA_THRESHOLD
```

This ensures the robot stops at exactly the configured follow distance.

---

## 7. Lateral Control

```python
msg.x
```

contains the smoothed horizontal centroid of the target person, allowing the robot to rotate and keep the person centered.

---

## 8. Fallback / Safety Conditions

The robot stops and returns to **SEARCHING** if:

- The person is lost
- Distance exceeds `MAX_FOLLOW_DIST_M`
- Distance is below `MIN_FOLLOW_DIST_M`
- ReID similarity drops below threshold
